# Auto Theme
Automatically changes Windows colors between morning and night.
(I made this software for myself (lol))

# How to use?
Simply download the Exe file from [Latest Release](https://github.com/dev-hiroki/AutoTheme/releases/latest) and open it.

# To register for startup
Just place the Exe file in %appdata%\Microsoft\Windows\Start Menu\Programs\Startup. It will open automatically at the next startup.
